# dotfiles
My dotfiles for X window server, bash, neovim

## Configs
- Xorg
	- Map `Capslock` to `ESC`
	- Increase keyboard key repeat rate
- Neovim
	- onedark theme
	- fzf
	- nerdtree
	- devicons
	- coc completion
- Bash aliases
	- Use `exa` instead of `ls`

## Dependencies
### For (neo)vim
- ctags (for tagbar)
- Fira Mono nerd font (for devicons)
- ripgrep (for fzf)
### For bash alias
- exa
